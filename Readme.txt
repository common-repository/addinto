AddInto is a bookmarking and sharing button that helps website publishers and bloggers spread their content across the web by making it easy for visitors to bookmark and share content to their favorite social destinations.

== Description ==

This plugin allows you to add the button below items in your WordPress blog. Some options were included in the administration for more customization :

* Many logos to choose from.
* Choose services to use in dropdown menu & window popup
* Alignment: left, center or right.
* Display area: Pages, archives, home, categories.

== Installation ==

1. Upload `/addinto/` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. (Optional) Customize the plugin in the Settings (or Options) > AddInto menu
